### Instruction
1. install docker
2. copy .env file
3. run `yarn docker-build` after each update
4. run `yarn docker-start` for reload

### DOCKER

Build docker
```$xslt
yarn docker-build
```

Start docker
```$xslt
yarn docker-start
```

Stop project
```$xslt
yarn docker-stop
```
